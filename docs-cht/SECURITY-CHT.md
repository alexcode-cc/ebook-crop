# 安全政策

## 支援版本

| 版本 | 支援狀態     |
|------|--------------|
| 1.6.x | 目前支援     |
| 1.5.x | 支援         |
| 1.4.x | 支援         |
| 1.3.x | 支援         |
| 1.2.x | 支援         |
| 1.1.x | 支援         |
| 1.0.x | 支援         |
| < 1.0 | 不支援       |

## 回報漏洞

若發現安全漏洞，請透過 [GitHub Issues](https://github.com/alexcode-cc/ebook-crop/issues) 回報，並在標題註明 `[Security]`。

請勿公開揭露未修復的漏洞。
